# MyMapAPP

## 概要
GoogleMapに地点や写真等を登録し、オリジナルの地図を作成するwebサービスです。<br>
「公園」「パン屋」等の区分を設定することにより、ジャンル毎の地図を作成します。

https://hrtada.github.io/mymap_app/

## 開発環境
＜フロントエンド＞<br>
・JavaScript <br>
　フレームワーク：Vue.js<br>
 
 ＜サーバーサイド＞<br>
・Node.js<br>
・SQlite3（Glitchi）<br>
・Firebase Strage（画像の保存に使用）<br>
・Firebase Authentication<br>
